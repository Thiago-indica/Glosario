Glosario de Programacion


A

Algoritmo: Un conjunto finito y ordenado de instrucciones claras y precisas para resolver un problema o realizar una tarea específica. Es la base de cualquier programa informático.

B

Bug (Error): Un defecto o fallo en el código de un programa que causa un comportamiento inesperado o incorrecto.

C

Compilador: Un programa que traduce el código fuente (escrito por el programador en un lenguaje de alto nivel) a código máquina (lenguaje que la computadora entiende directamente). Si encuentra errores de sintaxis, los reporta.

Computación: El estudio de los procesos algorítmicos que describen y transforman información. Incluye el diseño y construcción de hardware y software, el procesamiento, estructuración y gestión de información.

Código Fuente: El conjunto de instrucciones escritas por un programador en un lenguaje de programación (como C) antes de ser compilado o interpretado.

Comentario: Texto dentro del código fuente que es ignorado por el compilador o intérprete. Se utiliza para explicar el código, hacerlo más legible y documentar su funcionamiento.

Condicional (Estructura): Una estructura de control que permite al programa tomar decisiones. Ejecuta un bloque de código si una condición es verdadera y, opcionalmente, otro bloque si es falsa

Constante: Un valor que no puede ser modificado durante la ejecución de un programa.

Control de Flujo: La secuencia en que las instrucciones de un programa se ejecutan. Las estructuras de control (secuenciales, condicionales, repetitivas) alteran este flujo.

D

Depuración (Debugging): El proceso de encontrar y corregir errores (bugs) en el código de un programa.

E

Estructura de Datos: Una forma particular de organizar y almacenar datos en una computadora para que puedan ser usados de manera eficiente (ej. arrays, listas, pilas, colas).

F

Función (Procedimiento/Subrutina): Un bloque de código reutilizable que realiza una tarea específica. Las funciones ayudan a organizar el código, hacerlo más modular y fácil de mantener.

I

IDE (Integrated Development Environment - Entorno de Desarrollo Integrado): Un software que proporciona herramientas completas para el desarrollo de software, incluyendo un editor de código, un compilador/intérprete, un depurador y otras herramientas útiles.

L

Lenguaje de Bajo Nivel: Un lenguaje de programación cercano al hardware de la computadora, como el lenguaje ensamblador o el código máquina. Es más difícil de leer y escribir para los humanos.

Lenguaje de Alto Nivel: Un lenguaje de programación diseñado para ser más fácil de leer y escribir para los humanos, utilizando una sintaxis más parecida al lenguaje natural (ej. C, Python, Java). Necesitan ser compilados o interpretados.

Lenguaje de Programación C: Un lenguaje de programación de alto nivel (aunque cercano al bajo nivel) de propósito general. Es muy eficiente, potente y ofrece un gran control sobre el hardware, lo que lo hace popular para sistemas operativos, software embebido y aplicaciones de rendimiento crítico.

P

Programación: El proceso de diseñar, escribir, probar y mantener el código fuente de programas de computadora.

S

Sintaxis: El conjunto de reglas que define la forma correcta de escribir instrucciones en un lenguaje de programación. Un error de sintaxis impide que el programa compile o se ejecute.

V

Variable: Un espacio de almacenamiento con nombre en la memoria que puede contener un valor que puede cambiar durante la ejecución de un programa.
