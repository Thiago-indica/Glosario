Integrantes: Ramiro Perez, German lucero, Nadiea lucena, Thiago Muñiz, Lautaro Massa.

Base de datos: lugar en el que se almacenan todos los datos recolectados por una pagina.

SQL: es un lenguaje que ayuda a la toma de dicisiones.

DER: Diagrama de entidad-relacion es una forma de administrar una base de datos en la cual diagramas por entidades con atributos y lo relacionas con otras entidades.

entidad: es una "caja" la cual esta compuesta por atributos los cuales la diferencian de los demas.

relacion: como o que hace con otra entindad.

Atributos: caracteristicas que definen a una entidad.
